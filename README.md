# AryanLog

Personal blog platform built with React + TypeScript + Node.js + MongoDB.

## Structure

- `backend/` - Express.js + TypeScript API
- `frontend/` - React + Vite + TypeScript + Tailwind

## Development

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

Create `.env` files based on `.env.example` in each directory.
